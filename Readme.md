A brief description of your project.
This project is about creating a personal blog where users can write their own posts and see them displayed on the screen. Users can also edit or delete their posts whenever they want.

Instructions on how to run the application (if anything beyond opening index.html in a browser is needed).

Add a post by entering the title and content, then click submit.
After submitting, the post will appear on the screen.
If you want to delete a post, click the Delete button.
If you want to edit a post, click the Edit button, update the content, and it will be shown on the screen.

A reflection on your developmenwt process, challenges faced, and how you overcame them.
Any known issues or features not implemented.

While working on this project, I faced some challenges with handling forms and user input. At first, it was a bit confusing to manage everything, so I decided to use Bootstrap to make the UI easier and cleaner.

I also watched a few YouTube videos to understand how to implement the logic properly. One of the main difficulties I faced was adding functionality after using event listeners and displaying the data on the screen. I solved this by practicing more and breaking the problem into smaller parts.
